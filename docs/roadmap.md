# Scout Roadmap

> Scout is a local AI desktop sidekick that understands what is on your screen and provides contextual guidance.

---

# Vision

Build an AI assistant that can understand anything happening on a user's screen and provide intelligent, context-aware assistance without requiring the user to constantly explain what they're doing.
